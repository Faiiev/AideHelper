## 简介
一款专门为AIDE量身打造的Xposed插件，提供个性化选择。

如果你在VirtualXposed环境下打开AIDE很慢，那么去清除VirtualXPosed的数据，在重新使用就好多了

## 功能介绍

- 解锁专业版

- 禁用自动旋转

- 符号栏调整

- 搜索结果界面调整

- 去除Aide的icon

- 中文补全

- Gradle补全拓展

- 版本切换
